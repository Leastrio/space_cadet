# SpaceCadet
[![discord badge][]][discord link]

SpaceCadet is a **WIP** discord wrapper for the elixir programming language

# Features
## Multi Bot Support
SpaceCadet supports multiple bots running in the same Elixir app, all bots having seperate shard connections, ratelimiter processes, and cache processes.

## 

# Installation

The package can be installed
by adding `space_cadet` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:space_cadet, github: "Leastrio/space_cadet"}
  ]
end
```

Documentation can be found at <https://hexdocs.pm/space_cadet>.
